*“A poor workman blames his tools, but a great one knows how to choose them wisely.”*

Just FYI - I use a Mac.

### Package Managers
- [Homebrew](https://brew.sh/) - MacOS
- [npm](https://www.npmjs.com/) - Javascript
- [pip](https://pypi.org/project/pip/) - Python

### Shell
- [iTerm2](https://iterm2.com/) - Feature-rich Terminal
- [oh-my-zsh](https://ohmyz.sh/) — Manage ZSH configuration
- [zsh-syntax-highlighting](https://formulae.brew.sh/formula/zsh-syntax-highlighting) — ZSH configuration, highlights command shell syntax errors
- [zsh-autosuggestions](https://formulae.brew.sh/formula/zsh-autosuggestions) — ZSH configuration, suggests commands based on history
- [zsh-autocomplete](https://formulae.brew.sh/formula/zsh-autocomplete) — ZSH configuration, real-time type-ahead completion in the shell

### Version Control
- [Git](https://git-scm.com/)
- [Sourcetree](https://www.sourcetreeapp.com/) - git GUI

### Code Editor
- [VSCode](https://code.visualstudio.com/) - Has plenty of nice extensions

### Languages
- [Python](https://www.python.org/downloads/macos/)
- [Node.js](https://nodejs.org/en)
- [Go](https://go.dev/doc/install)

### Whiteboarding
- [Excalidraw](https://excalidraw.com/) — I like the seamless drawing experience, great for quick prototyping and system design demos

### Database GUI
- [Studio 3T](https://studio3t.com/) - I use this for MongoDB, it offers a feature-rich UI
- [MongoDB Compass](https://www.mongodb.com/products/tools/compass) - An alternative to above
- [Medis](https://getmedis.com/) — For viewing Redis DB
- [DataGrip](https://www.jetbrains.com/datagrip/) - Versatile, requires separate driver installations for specific databases, mainly used this to manage SQL databases
- [MySQL Workbench](https://www.mysql.com/products/workbench/) - Not as great as DataGrip, but more straightforward to set up

### API Testing
- [Postman](https://www.postman.com/) — A standard must-have for testing API calls
- [Bruno](https://www.usebruno.com/) — Nice alternative to above, has git integration

### Containers
- [Docker Desktop](https://docs.docker.com/desktop/) — Nice GUI for playing around with images and containers

### Misc
- [Rectangle](https://rectangleapp.com/) — For managing app windows using keyboard shortcuts
- [Epoch Time Converter](https://www.epochconverter.com/) - For generating or converting timestamps
- [JSON Stringify](https://jsonformatter.org/json-stringify-online) - For stringifying JSON objects
- [HTML Editor](https://www.w3schools.com/html/tryit.asp?filename=tryhtml_editor) - For previewing HTML code
- [Base64 Encode and Decode](https://www.base64decode.org/) - For encoding or decoding strings in base64
- [Regex101](https://regex101.com/) - Verify and test your regex
- [ChatGPT](https://chatgpt.com/) - Self-explanatory
- [Copilot](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot) - Free to use if you register as a student. If your company has purchased a license for it, good for you!
